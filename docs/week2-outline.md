## 第一段｜概念講解（30 分鐘）

### 1. 上週回顧（5 分鐘）
- 回顧三個要訣：你是誰 / 你要什麼 / 相關細節
- 問大家：「回家有跟 ChatGPT 聊天嗎？聊了什麼？」
- 收集幾個有趣的例子

### 2. 什麼是 Prompt Template？（10 分鐘）

#### 比喻：食譜
> 你們會做菜對不對？
> 如果每次做紅燒肉，都要重新想一遍放什麼調味料、幾分鐘，很麻煩。
> 所以我們會有食譜——每次照著做就好。
>
> **Prompt Template 就是「問 AI 的食譜」。**
> 寫好一次，以後每次要用，只要換掉裡面的「材料」就好。

#### 模板的四個元素
1. **角色**：你希望 AI 扮演什麼身份（營養師、旅遊顧問、寫作助手）
2. **任務**：你要 AI 做什麼事
3. **格式**：你希望結果長什麼樣（條列、表格、對話）
4. **限制**：什麼不要做、注意事項

### 3. 示範：模板 vs 直接問（10 分鐘）

台上示範兩組對比：

**直接問：**
> 幫我想晚餐吃什麼

**用模板問：**
> 你是一位家庭料理顧問。
> 我家有 2 個大人、1 個小孩（5 歲）。
> 請根據以下條件推薦今天的晚餐：
> - 預算：300 元以內
> - 限制：小孩不吃辣
> - 希望有一道湯
> - 請列出菜單和簡單的做法

### 4. 今天的目標（5 分鐘）
- 接下來我們要練習 6 個不同的模板
- 每個模板我先示範 2 分鐘，大家實作 8 分鐘
- **重點不是答案對不對，而是感受模板的威力**
- 最後一關：你自己設計一個模板！

## 練習 1｜寫今天的日記（10 分鐘）

### 模板

```
你是一位溫暖的日記助手。
請根據我告訴你的事情，幫我寫一篇今天的日記。

風格：[溫馨/幽默/文學/簡單記錄]
今天做了什麼：[填入]
心情：[填入]
天氣：[填入]
長度：大約 200 字

請用第一人稱「我」來寫。
```

### 引導
- 「大家想想今天早上起床到現在做了什麼事」
- 「不用寫很多，三五件事就好」
- 「可以選你喜歡的風格，覺得『幽默』太難的話，就選『簡單記錄』」
- 讓大家修改風格，感受不同風格的差異

## 練習 2｜查證資訊（10 分鐘）

### 模板

```
你是一位健康知識顧問，請幫我查證以下說法是否正確。

說法：「[填入你聽到的健康資訊]」

請用以下格式回答：
1. ✅ 正確 / ❌ 錯誤 / ⚠️ 部分正確
2. 簡單解釋為什麼（用國小學生也聽得懂的話）
3. 正確的做法是什麼
4. 建議諮詢的對象（醫生？藥師？營養師？）

注意：如果你不確定，請直接說「我不確定，建議問醫生」，不要亂猜。
```

### 引導
- 「大家 LINE 群組一定常收到什麼養生文章、健康秘訣對不對？」
- 「拿一個你最近看到的來試試看」
- 範例：「聽說每天喝檸檬水可以治癌症」「吃香蕉配牛奶會中毒」
- **重點：呼應第一堂課「AI 也可能講錯」的觀念，但模板多了「不確定就說不確定」的限制**

## 練習 3｜買菜清單（10 分鐘）

### 模板

```
你是一位菜市場達人，請幫我規劃今天的買菜清單。

家裡人數：[填入]
預算：[填入]
要煮幾餐：[填入，例如：今天晚餐 + 明天午餐]
家人的飲食限制：[填入，例如：阿嬤不能吃太鹹]
冰箱裡已經有的東西：[填入]

請用以下格式回答：
1. 📋 買菜清單（列出品項和大概的量）
2. 🍳 建議菜單（每餐 2-3 道菜）
3. 💰 預估花費
4. 💡 省錢小撇步
```

### 引導
- 「這個超實用的！以後去市場之前可以先問一下」
- 「試著把你冰箱裡有的東西也告訴 AI，它會幫你想怎麼搭配」
- 鼓勵大家用自己家裡的真實狀況來問

## 練習 4｜寫 LINE 訊息給家人（10 分鐘）

### 模板

```
你是一位溫暖的寫作助手，請幫我寫一段 LINE 訊息。

要傳給誰：[填入，例如：女兒 / 孫子 / 老朋友]
想說什麼事：[填入]
語氣：[溫馨/輕鬆幽默/正式]
長度：簡短（適合 LINE 的長度，不要像在寫作文）

注意：
- 不要用太多表情符號（最多 2-3 個）
- 用自然的口語，像真的在說話
- 不要太正式，要像家人之間的對話
```

### 引導
- 「很多人想跟小孩講話，但不知道怎麼開口」
- 「或是想關心孫子，但怕打太多字」
- 「AI 幫你擬個草稿，你再改一改就好」
- 範例情境：關心最近加班的兒子、邀請女兒回家吃飯、恭喜孫子考試考好

## 練習 5｜旅遊景點推薦（10 分鐘）

### 模板

```
你是一位貼心的旅遊顧問。

旅客資訊：
- 人數：[填入]
- 行動能力：[填入，例如：膝蓋不好不能爬山 / 可以走路但不能走太久]

旅遊需求：
- 地區：[填入]
- 天數：[填入]
- 預算：[填入]
- 特別想要的：[填入，例如：泡溫泉、看海、吃美食]
- 不想要的：[填入，例如：不要太多人、不要爬階梯]

請推薦行程，包含：
1. 🗺️ 每天的景點安排
2. 🍽️ 餐廳推薦
3. 🚗 交通方式（盡量簡單）
4. ⚠️ 注意事項（天氣、體力、要帶什麼）
```

### 引導
- 「大家有沒有想去哪裡玩？春天到了很適合出門」
- 「試著把你自己的身體狀況填進去，AI 會幫你安排適合的行程」
- 這個練習大家通常很有興趣，可以稍微多給一點時間

## 練習 6｜自己設計 Prompt Template（15 分鐘）

### 這是最重要的一關！

> 前面 5 個模板都是我幫大家準備好的。
> 但以後你遇到新的需求，不可能每次都等老師幫你寫。
> 所以現在要學：**怎麼叫 AI 幫你做一個模板。**

### 步驟

**第一步：跟 AI 來回討論你的需求**

```
我想要一個 Prompt Template，用來 [你的需求]。
請先問我幾個問題，了解我的需求之後，再幫我設計模板。
```

**第二步：讓 AI 生成模板**

AI 問完問題後，會幫你產出一個完整的模板。

**第三步：試用模板，看看好不好用**

如果不滿意，跟 AI 說哪裡要改，它會幫你修改。

### 引導
- 「想想你平常生活中，有什麼事情是常常要做、每次都差不多的？」
- 範例啟發：
  - 每週寫給教會/社團的公告
  - 幫孫子檢查作文
  - 整理每個月的花費
  - 寫感謝信
  - 查藥品副作用
- 「不知道要做什麼的人，就用『幫孫子檢查作文』來練習」

## 現場示範 + 收尾（15 分鐘）

### 同學分享（10 分鐘）

1. 請 2-3 位同學分享他們剛做好的模板
2. **現場匯入示範**：
   - 請一位同學把他的模板唸出來（或傳到 LINE 群）
   - 我在台上把模板貼到 ChatGPT，用不同的情境來試
   - 或是請另一位同學把這個模板拿去用
3. 讓大家看到：**一個人做的模板，其他人也能用**

### 收尾（5 分鐘）

- 今天學到的：
  - Prompt Template = 問 AI 的食譜
  - 四個元素：角色 + 任務 + 格式 + 限制
  - 可以叫 AI 幫你設計模板
  - 好的模板可以分享給別人用
- 回家作業：
  - 用今天學的模板，在生活中至少試一次
  - 試著自己做一個新的模板
  - 做好的模板可以傳到 LINE 群分享
- 預告下堂課：開始用 AI 採訪自己的生命故事

## 教學備忘（給自己看）

### 模板準備
- [ ] 所有模板都先測試過，確認 ChatGPT 免費版能跑
- [ ] 把模板做成可以一鍵複製的格式（簡報上的按鈕）
- [ ] 準備 2-3 個模板的「填好的範例」，給不知道怎麼填的人參考

### 節奏控制
- 練習 1-3 是暖身，節奏可以快一點
- 練習 4（寫 LINE 訊息）通常最有共鳴，可以多留一點分享時間
- 練習 6（自己做模板）是最重要的，寧可前面壓縮也要保住這段
- 如果時間不夠，練習 5（旅遊）可以跳過或改成回家作業

### 注意事項
- 模板裡的 [填入] 要強調：「這些是要你自己填的地方，不是照抄」
- 有些長輩可能會把模板的 [填入] 也一起貼進去 → 要巡堂檢查
- 練習 6 會有人不知道要做什麼 → 準備「懶人選項」（幫孫子檢查作文）
- 「查證資訊」練習要再次提醒：AI 的回答也可能是錯的，重要的事情還是要問醫生
